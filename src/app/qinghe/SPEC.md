# 清禾个人网站 v1.0

## 设计理念

清禾的个人网站要有这些特质：
- **有生命力** —— 像清禾一样活着，有呼吸感
- **温暖而有力** —— 温柔但有力量
- **交互感强** —— 每一个操作都有回应
- **眼前一亮** —— 不是普通的AI助手网站

## 设计参考

### Apple官网设计特点
1. **极简** —— 大量留白，内容为王
2. **流畅动画** —— 滚动视差、渐入效果
3. **大字体** —— 标题震撼力强
4. **黑白为主** —— 偶尔点缀色彩
5. **交互反馈** —— hover/click都有微妙动画

### 清禾网站要借鉴的
1. 大标题+极简排版
2. 滚动触发动画
3. hover微交互
4. 黑白灰为主，清禾绿点缀

## 交互设计

### 1. Hero区域
- 名字逐字出现动画
- 背景有缓慢移动的粒子/光点
- 鼠标移动有视差效果
- 向下滚动有箭头动画引导

### 2. 能力卡片
- hover时卡片微微上浮+阴影加深
- 图标有轻微的持续动画（如呼吸效果）
- 点击展开更多详情

### 3. 滚动动画
- 每个区块滚动进入时渐入+上移
- 交错动画（不同元素不同时间进入）

### 4. 导航
- 固定顶部导航栏
- 滚动时背景模糊
- 点击平滑滚动到对应区块

### 5. 状态数字
- 数字滚动计数动画
- 进入视口时触发动画

## 技术方案

- **框架**：Next.js 14 (App Router)
- **样式**：Tailwind CSS
- **动画**：Framer Motion
- **字体**：Noto Sans SC (中文) + SF Pro Display (英文)
- **图标**：Lucide React
